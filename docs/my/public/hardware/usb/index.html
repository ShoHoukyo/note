iPhone​

使用说明: 按图连接iPhone 不需要安装任何app（如果包装上推荐app,也不要安装） 把需要打印的文件*copy(コピー)*到USBフラッシュメモリ 拔下USBフラッシュメモリ插入机器打印（这样一般的店都可以打印） 打印完了之后删除文件

注意： USBフラッシュメモリ是不可靠的媒体，无论多贵的都有可能

  • 遗失（因为太小），而数据没有加密会造成数据泄露
  • 因各种原因（温度，湿度等等）丢失数据 所以请务必使用完了之后删除文件，也不需要买大容量的，坏了买新的就可以 （手机更可靠，但是本质上来讲手机也不是100%可靠的，重要数据还是要备份，简单的办法就是用apple的云）
