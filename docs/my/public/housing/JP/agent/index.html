不要盲目寻找不认识的中国人中介，也不要盲目相信日本人中介。尽量找靠谱的
日本中介或者有靠谱人士介绍的中国人中介

其实完全陌生的中介，无论是中国人还是日本人，都最好留个心眼。
我自己就因为对日本中介太过相信，最后被坑，差点流落街头。所以真的呼吁大
家不要觉得日本的服务都很好很有信誉就轻易相信。房地产业界绝对是一朵奇葩
的！在你签订租房的条约之前，只是咨询阶段，所以不管怎么出尔反尔他们都是
不违法的。

最后我现在的房子就是找的比较熟的做中介的朋友。其实知根知底的中国人中介
其实是比日本中介负责不知道多少倍的。当时朋友每天反复给保证公司和管理公
司打电话催，不断地帮我跑，短短四五天就让我搬到了新家，帮了大忙。如果是
在东京找房子的朋友，觉得我可以信赖的请单独私信询问，可以把这个帮忙找房
子的朋友介绍给你（他们也给学生介绍比较便宜的学生寮）。尤其是还没来东京，
对日语没信心或者想要提前看房尽快入住省点旅馆费的朋友也比较推荐。(不过，
对我半信半疑的，害怕被坑被骗的就不要联系我了，完全可以自己从其他途径找
其他人。解释什么的是浪费彼此的时间。)



关于找房子的经验，上面很多答主都说的非常明白了。反正在日本租房是一件非
常麻烦和费精力的事情，找房一定要做好打长期站的准备，不到拿到合约绝不能
放松。这里说点关于中介公司的经验。个人觉得选择房产中介公司非常重要。特
别需要确认的是中介公司是不是房产的管理公司。日本的很多房产仲介公司可以
分为两种，一种是自己管理房产同时做租赁，另一种是彻底的皮包公司，只是掌
握了空房的信息。有的管理公司虽虽然也有自己做中介服务，但是把情报共有给
其他公司可以减少房屋的空置率，所以这些公司之间的空房资讯是共享的，但实
际签约权都属于房屋的管理公司，所以如果你找到的是一家纯粹的皮包公司，他
们只是帮助你和管理公司进行联系，任何实质性的问题他们都需要得到管理公司
的同意才可以，这就意味着你多了一个传话的人，增加了得到回复的时间。为什
么重要。因为这决定了他们有没有决定权。如果是皮包公司，最后即使他们想租
给你，但是管理公司说不，他们也是一点办法都没有的。要是真碰到这种状况，
你看房花的时间和精力都浪费。我就有过最惨痛的教训。刚准备从大阪搬到东京
的时候，专门休假来找房。看中了一家很不错的房子，中介也觉得我的资格绝对
没问题，最后中介打电话和房产管理公司联系，对方听说是中国人后一定要和我
的单位签法人合同才同意租。


那下面就涉及到如何识别是不是皮包公司。最重要的就是看当你问到比较关键的
问题的时候，他是不是每次都必须打电话问别人这一点上。以我的经验最简单的
可以从拿房门钥匙这件事来判断。如果你看好了几间房想实地看房，中介如果马
上说好的带你看，并从公司拿出钥匙带你走，那就说明他们肯定是直接管理房产
的公司。因为有的时候钥匙是密码锁，或者是放在房子的水表箱的密码盒里的。
那就看他是否需要打电话向别的公司问密码。当然中介不可能记得所有密码，即
使使本公司的管理房产可能也需要向别人询问，但是他打电话的时候是跟本公司
人对话还是跟外公司人对话，从态度上是很容易识别的。不过可以说你能够在网
上找到的中介基本上都是皮包公司。关键是不要找到没良心的。如果时间比较充
裕，建议是到想要住的地段找一些不是连锁的，小的或是很旧的中介公司（日文：
町の不動産屋）。因为这种公司很多都是住在当地的人开的，老板年级都会比较
大。他们很多人和房东可能从小认识，是朋友，可以作为房东的全权代理。那礼
金，保证金，房租什么都好谈。聊天聊的开心了给你中介费大优惠都是有可能的。
前提是你的日语要好。关于房租砍价，我觉得大于5000日元的降价是基本没有希
望的。为什么，一个月5000日元意味着房东一年损失6万，你多租一年他多损失6
万。你的房租如果正好是6万，他很可能愿意空放两个月找个愿意出6万的日本人。
因为他还很可能不愿意冒险把房子租给是外国人的你。砍价当然是必须的，但是
不要在房租上多做文章，尽量砍掉礼金和保证金。皮包公司是不可能和你谈价的，
因为没有决定权，如果你没有办法找到的是这种公司不要浪费精力在杀价上了。
租房的是弱势群体，没有谈价的资本的。在掌握的情报不对等的状况和找房有时
间限制的情况下，不要希望中介能够象日本的别的服务行业一样把你当上帝。饭
馆你要天天下，找房他希望你月月找他你也不会的，你和他的交易这辈子不会超
过3次的。我来了15年，搬过7次家，没用过一次同样的中介。得到的经验是皮包
中介公司，他能赚你的是绝对不会放弃的。


中国人中介谈不下来（不租外国人的房子），日本人中介搞得定的例子我也经历过，如果很在意的话，日本不动产的中国店员是你的一个选择
